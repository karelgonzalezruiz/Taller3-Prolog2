# Taller3-Prolog2
Resolución del taller 3.

Que hice en este taller:

* **Comparaciones Aritméticas:** Regla simple para determinar el número máximo entre dos valores.
* **Consultas de Tiempo y Fechas:** Uso de hechos históricos (reinados) y reglas lógicas para verificar si un monarca gobernó en un año exacto o de forma ininterrumpida durante un periodo.
* **Lógica Condicional y Datos Estructurados:** Creación de una base de conocimientos gramatical (tiempos, personas, números) y uso de la estructura If-Then-Else (`-> ;`) para conjugar dinámicamente el verbo inglés "to be".
